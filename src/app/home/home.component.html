What is ?

CanYaDAO is the decentralised autonomous organisation that supports the CanYa ecosystem.

CanYa members can choose to enter the DAO by staking tokens and then selecting from a number of daily tasks to help run the ecosystem. Each task once complete will pay out $CAN tokens to the member and important tasks are prioritised with increasing amounts of $CAN. Tasks include curation, user and provider support tickets, dispute resolution, categorisation and tagging. Complex and important tasks (such as dispute resolution) are decided upon by a consensus decision from random DAO members.

DAO members can report other DAO members that act maliciously and earn their stake if the malicious member is determined by consensus to be malicious.

The outcome is an organisation that can run the CanYa ecosystem in a permissionless, decentralised and autonomous way.
